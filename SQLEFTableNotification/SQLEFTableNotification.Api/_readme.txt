RestApiNEx/ApiNCoreEx is Visual Studio extension for building .NET(5/6/7+) and .NET Core REST API services.
Generate 100s lines of code on a single click and save hours of searching Web and testing for the right REST API solution.
A great start for creating REST API service based on latest .NET(5/6/7+) and .NET Core. These extensions work with all Visual Studio editions including free Community VS edition.
